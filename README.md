# NFC De-Scrambler

**Retro-futuristic NFC encryption/decryption tool for mobile devices.**

## What It Does

▣ **Encrypt** messages and URLs  
▣ **Decrypt** NFC tags and manual codes  
▣ **Time-based security** - keys change every 5 minutes  
▣ **Works offline** - no server required  

## Mobile-First Design

Built specifically for mobile use with a retro LCD aesthetic inspired by District 9 and Borderlands tech.

## Quick Start

1. **Open `index.html` in Chrome on Android**
2. **Encrypt Mode**: Create encrypted codes for NFC tags
3. **Decrypt Mode**: Scan tags or enter codes manually
4. **Orange LCD display** shows all results

## NFC Support

- **Android Chrome**: Full NFC scanning capability
- **Other browsers**: Manual input mode available
- **Test mode**: Built-in simulation for testing

## Files

- `index.html` - Main app
- `styles.css` - Retro LCD styling  
- `crypto.js` - Encryption engine
- `app.js` - App logic
- `nfc-handler.js` - NFC integration
- `test.html` - Test suite (optional)

**Just copy these files to your phone and open in Chrome. That's it.**
